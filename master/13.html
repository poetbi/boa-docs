boaPHP开发手册v5.5

系统配置

config.php

'VERSION' => '',
'NAME' => '',
'CHARSET' => '',
'SALT' => '',
'DEBUG' => ,
'LANGUAGE' => '',

配置

配置项默认值类型说明
VERSION5.5string必须；系统版本
NAMEBoaSoft.Topstring必须；系统默认名称
CHARSETUTF-8string必须；系统字符集
SALTboa2020string必须；系统各种加密盐度值，建议设置复杂一点
DEBUGfalsebool可选；是否开启调试模式
LANGUAGEzh-cnstring可选；默认语言包

路径常量

两个必须常量是必须在入口文件中定义的，可选常量可以在config.php中定义，默认为系统自动。如实例资源分离部署到独立服务器，则可以为WWW_RES定义一个http开头的URL路径

常量定义说明
BS_ROOT必须系统根目录的绝对路径，如：D:/boasoft.top/
BS_WWW必须当前实例目录（如www）的绝对路径，如：D:/boasoft.top/www/
BS_VAR可选当前临时目录（如var/www）的绝对路径，如：D:/boasoft.top/var/www/
BS_BOA可选内核目录（如boa）的绝对路径，如：D:/boasoft.top/boa/
BS_MOD可选模块目录（如mod）的绝对路径，如：D:/boasoft.top/mod/
WWW可选当前实例目录（如www）的相对路径或URL路径（以http开头），如：/ 或 http://xxx.boasoft.top（手动配置）
WWW_RES可选当前实例资源目录（如www/res）的相对路径或URL路径（以http开头），如：/res/ 或 http://res.boasoft.top（手动配置）
WWW_FILE可选当前实例文件目录（如www/file）的相对路径或URL路径（以http开头），如：/file/ 或 http://file.boasoft.top（手动配置）

注意

不支持 cfg() 动态配置
