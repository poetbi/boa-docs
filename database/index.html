数据库

主要特性

  • 支持pdo和mysqli等驱动
  • 支持tcp、socket和pipe等连接方式
  • 支持单例和分布式模式
  • 支持数据库持久化连接
  • 支持原生SQL查询
  • 支持链式构造查询
  • 支持预处理批量查询
  • 支持事务型和锁机制
  • 支持同时操作多数据库
  • 支持多表关联查询
  • 支持数据库日志记录

db 为 database 别名，boa::db() 等同于 boa::database()

相关API

  • database （数据库）
    • builder （构造器）
    • stmt （预处理）
    • driver.mysqli （mysqli驱动）
    • driver.pdo （pdo驱动）
  • model （模型基类）
