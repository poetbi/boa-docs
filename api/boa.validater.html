validater类
版本 1.0+
命名空间 boa
文件 boa/validater.php
说明 变量验证类，配置：'FILTER' => '',
配置
配置项 默认值 类型 说明
FILTER string 如果设置，自动变量验证时如未设置过滤，则使用此值，比如：htmlspecialchars
目录
方法 说明
execute() 执行一条验证
方法

execute()

↑
说明
执行一条验证
参数 必须 默认值 类型 说明
$key Y string 键
$val Y string 值
$rule Y array 验证规则
返回值
返回处理后的值（string 或 array），验证不通过会报错
示例 
/*----variable-------*/
$rule = [
	'label' => '用户名',
	'check' => 'required&is_name',
	'filter' => 'htmlspecialchars'
];
/*----variable-------*/

$v = boa::validater();
$username = $v->execute('username', "test's name", $rule);
