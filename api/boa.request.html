request类
版本 1.0+
命名空间 boa
文件 boa/request.php
说明 请求处理类
目录
方法 说明
__construct() 初始化
__get() 魔术方法，以属性的方式获取请求参数，等同于 v($k)
__set() 魔术方法，以属性的方式为请求参数赋值
validate() 手动显式验证所有用户变量
raw() 得到未经处理的原始用户数据
v() 访问请求参数，并执行自动验证

v()

↑
说明
访问请求参数，并执行自动验证
参数 必须 默认值 类型 说明
$key Y string 参数键名，将依次从环境var、 $_POST、 $_GET、 $_COOKIE中取值，然后执行自动验证
返回值
返回参数值（string 或 array）
示例 
//boa::env('var.name')
$name = $this->request->v('name');

//$_POST['name']
$name = $this->request->v('name');

//$_GET['name']
$name = $this->request->v('name');

//$_COOKIE['name']
$name = $this->request->v('name');
