server ['127.0.0.1', 11211] array 服务器池，支持多组服务器（二维数组），3个元素分别表示：
  • 0：主机，必须；socket方式如 /path/memcached.sock
  • 1：端口，可选；默认 11211，为0时使用socket方式连接
  • 2：服务器权重，可选；
