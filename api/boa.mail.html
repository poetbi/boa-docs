send()

↑
说明
发送邮件，成功产生info日志，失败产生error日志
参数 必须 默认值 类型 说明
$subject Y string 用邮件主题
$message Y string 邮件正文
$to null string 收件人，默认使用配置信息
返回值
成功返回0（int），失败返回错误码（int）
示例 
$mail->cfg('to', 'xxx@mail.com 张总'); // 收件人
$mail->send('你好','再见！');
