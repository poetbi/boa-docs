installer类
版本 1.0+
命名空间 boa
文件 boa/installer.php
说明 安装器类
目录
方法 说明
initlize() 初始构建、初始一个新模块或初始一个新实例
install() 安装模块，具体执行模块中安装器安装
uninstall() 卸载模块，具体执行模块中安装器卸载
方法

initlize()

↑
说明
初始构建、初始一个新模块或初始一个新实例
参数 必须 默认值 类型 说明
$www www string 初始一个实例（同时创建对应的临时目录）
$mod home string 初始一个模块
示例 
命令行中执行：
php boa -b //初始化构建
php boa -m test //创建test模块
php boa -w boasoft.top //创建boasoft.top实例

install()

↑
说明
安装模块，具体执行模块中安装器安装
参数 必须 默认值 类型 说明
$mod Y string 模块名
示例 
命令行中执行：
php boa -i demo // 安装demo模块

uninstall()

↑
说明
卸载模块，具体执行模块中安装器卸载
参数 必须 默认值 类型 说明
$mod Y string 模块名
示例 
命令行中执行：
php boa -u demo // 卸载demo模块
