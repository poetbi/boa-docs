encode()

↑
说明
压缩字符串
参数 必须 默认值 类型 说明
$data Y string 待压缩字符串
$type zlib string 压缩类型，可选值：raw, zlib, deflate, lzf, bzip2
$level -1 int 压缩级别，可选值：0-9，对lzf无效
返回值
返回压缩后的字符串（string），出错则报错
示例 
$res = $archive->encode($data);
