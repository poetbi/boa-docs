is_ajax()

↑
说明
判断是否AJAX请求
参数 必须 默认值 类型 说明
$param _ajax string ajax请求手动标示字段，如前端请求指定该字段，则表示是ajax请求
返回值
如果手动指定param值字段或包含HTTP_X_REQUESTED_WITH请求头返回true（bool），否则返回false（bool）
