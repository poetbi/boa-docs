boaPHP开发手册v5.5

安装运行

准备

boaPHP运行环境要求：PHP 7.0 - PHP 8.3

boaPHP手册（适合开发时查询），boaPHP教程（适合零基础学习）

安装

可以选择以下任一方式安装，新手建议选择集成环境（免配置开箱即用，内附零基础教程）：

  • 手动安装：通过 github 或 gitee 下载源码，解压即可
  • 集成环境：下载 boaServer，集成Nginx+MySQL+PHP+boaPHP+Demo，一键运行，轻松管理
  • Composer：命令行中执行 composer create-project boasoft/boaphp

    • Composer的安装、使用教程（中文 英文）及国内镜像（阿里云 镜像一 镜像二）

    运行

    安装后浏览器中访问：http://localhost （首次访问会进行初始化）

    鉴于安全性，建议将域名绑定到www目录，而非根目录

    Demo

    初次接触boaPHP的用户可以下载一个Demo模块（github 或 gitee），完整的演示了boaPHP的常见用法。下载后解压到mod目录下（demo模块），命令行中执行：

    	php boa -i demo

    安装后浏览器中访问：http://localhost/?m=demo

    命令行中 boa命令 位于www目录下，注意切换至www目录，更多参见 命令行用法
